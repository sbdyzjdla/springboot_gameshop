# :video_game: 스프링부트로 만드는 온라인 쇼핑몰

## 사용 기술

**Back-End** 
* Spring boot
* Srping MVC
* Srping data JPA
* Spring Security
* gradle

**Database**
* Maria DB

**AWS**
* EC2
* RDS
* S3

**CI**
* Jenkins

**API Service**
* Oauth2(Google, Naver, KaKao) Login (배포시에는 카카오만 동작)
* 카카오 우편번호 서비스
* 아임포트 결제 API 

**Front-End**
* Javascript
* jQuery
* Mustache
